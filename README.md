# ELIS

ELIS includes:

- A premium chat frontend (`/` root project, React + Vite)
- A production-ready backend API (`/backend`, Node.js + Express + MongoDB)

## Frontend

```bash
npm install
npm run dev
```

## Backend

```bash
cd backend
npm install
cp .env.example .env
npm run dev
```

For full backend setup and API docs, see [`backend/README.md`](backend/README.md).
